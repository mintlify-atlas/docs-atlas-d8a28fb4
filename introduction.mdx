# Welcome to Budget Bee

Budget Bee is a simple accounting tool designed to help you manage your income, expenses, subscriptions, budgets and more. Built with modern web technologies, it provides an intuitive interface for tracking your financial data.

## What is Budget Bee?

Budget Bee is an open-source accounting software specifically designed for non-accountants. It simplifies financial management by providing:

- **Transaction Tracking**: Easily record and categorize income and expenses
- **Multi-Account Support**: Manage multiple financial accounts in one place
- **Subscription Management**: Track recurring payments and subscriptions
- **Custom Dashboards**: Create personalized views with drag-and-drop widgets
- **Team Collaboration**: Work with your team through multi-tenant organizations
- **Import/Export**: Easily import and export data in CSV, Excel, and JSON formats

## Who is Budget Bee For?

Budget Bee is perfect for:

- **Individuals**: Personal finance management without the complexity of traditional accounting software
- **Freelancers**: Track income and expenses for multiple projects
- **Small Teams**: Collaborate with team members using organization features
- **Small Businesses**: Manage business finances with multi-user access and role-based permissions

## Technology Stack

Budget Bee is built with modern, production-ready technologies:

- **Frontend**: Next.js 16, React 19, TailwindCSS
- **Backend**: Better Auth for authentication, PostgreSQL for data storage
- **State Management**: Zustand, TanStack Query
- **UI Components**: Radix UI primitives
- **Data Visualization**: Recharts for beautiful charts and graphs

## Open Source

Budget Bee is open-source software released under the ISC license. You can:

- Self-host your own instance
- Contribute to the codebase
- Customize it for your needs
- Review the security and privacy features
