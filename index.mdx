<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-3xl font-bold dark:text-white text-gray-900 mb-3">Quick Start</h2>
  <p className="dark:text-gray-400 text-gray-600 mb-8">Get up and running with Budget Bee in minutes</p>
  
  <Steps>
    <Step title="Create your account">
      Sign up for Budget Bee and verify your email address to get started.
      
      ```bash
      curl -X POST https://api.budget-bee.site/auth/sign-up \
        -H "Content-Type: application/json" \
        -d '{"email": "you@example.com", "password": "secure_password"}'
      ```
    </Step>
    
    <Step title="Complete your profile">
      Set up your profile with your name and preferences. This helps personalize your Budget Bee experience.
      
      ```typescript
      const profile = await fetch('/api/auth/user/update', {
        method: 'POST',
        headers: { 'Authorization': 'Bearer YOUR_TOKEN' },
        body: JSON.stringify({
          name: 'John Doe',
          email: 'john@example.com'
        })
      });
      ```
    </Step>
    
    <Step title="Add your first transaction">
      Start tracking your income and expenses by adding transactions.
      
      <Accordion title="View example response">
        ```json
        {
          "id": "txn_123456",
          "amount": -45.99,
          "description": "Grocery shopping",
          "category": "Food & Dining",
          "date": "2026-03-05",
          "account_id": "acc_789"
        }
        ```
      </Accordion>
      
      Learn more in our [Transactions guide](/features/transactions).
    </Step>
  </Steps>
</div>
